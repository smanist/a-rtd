# a-rtd

`a-rtd` is shared tooling for a small family of Sphinx/MyST ReadTheDocs course-note repositories.

Version `0.1.0` provides:

- `a-rtd init --profile sphinx-myst-course --from-existing`
- `a-rtd check`
- `a-rtd diff`
- `a-rtd update`
- the `a_rtd.sphinx_ext` Sphinx extension

The first milestone is based on `rtd_skeleton` and preserves current production-repo conventions such as `requirements.txt` and repo-local Codex skills.
